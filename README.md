# KlaudWebsite
The official KlaudOS Website's source code with http server made in c.

## Run locally
```bash
# Clone this repository
$ git clone https://github.com/coldmayo/KlaudWebsite

# Go into the repository
$ cd KlaudWebsite

# Conjure website
$ ./make.sh

# Run
$ ./run.sh

Go to http://localhost:8080/
```
