Installation Page

>

The first Klaud based Operating System


Installation

Before you do anything, make sure you have the below installed:
make
gcc
nasm
qemu-system-x86

Download From Release

Download disk image and sh file from release on github
run: ./run.sh in the terminal

Download From Source

Clone repo: git clone https://github.com/coldmayo/KlaudOS
Get into folder: cd KlaudOS
Conjure disk image: make
Run Operating System in QEMU: ./run.sh
