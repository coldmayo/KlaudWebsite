Installation Page

The first Klaud based Operating System


Installation

Before you do anything, make sure you have the below installed:
make
gcc
nasm
qemu-system-x86

Download From Release

Download disk image and sh file from release

Make executables:
chmod +x run.sh

Run Operating System in QEMU:
./run.sh

Download From Source (Linux)

Clone repository

cd into folder

Make executables:
chmod +x make.sh
chmod +x run.sh

Conjure disk image:
./make.sh

Run Operating System in QEMU:
./run.sh
